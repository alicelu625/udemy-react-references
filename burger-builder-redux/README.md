## Burger Builder Application with Redux

Puts Redux to practice by modifying the Burger Builder to use Redux to manage states in a global store.

Application was developed along with the [React course on Udemy](https://www.udemy.com/course/react-the-complete-guide-incl-redux/).

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

In the project directory, you can run:
##### `npm install`

then

##### `npm start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
